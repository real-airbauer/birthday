A Discord bot that remembers birthdays and calculates the age from them

## Features

- Set your Birthday
- Show your Age
- Show when your birthday is
- Fully configurable


## Run Locally
Clone the project

```bash
  git clone https://github.com/real-airbauer/birthday
```

Go to the project directory

```bash
  cd birthday
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
